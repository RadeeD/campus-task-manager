# Campus Task Manager

A fullstack web application for university students to manage assignments and deadlines.

## Features

- JWT Authentication
- Task CRUD
- Deadline Tracking
- Statistics Dashboard
- Priority Management

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express.js

Database
- MySQL
